== Description ==

Seo 301 Meta is an easy and seo effective way to redirect your internal and external links to new posts, pages, categories and websites.
This plugin helps you to redirect all kind of links to other pages or websites and add 301 redirection which is very important for seo.
*Easiest Way to Redirect your Wordpress Links to Required pages, posts or custom links.*

Features include:
* Simple Control Panel easy to use for beginners and newbies.
* Wildcard support
* Redirect posts and pages that published, draft or deleted from edit page directly with 301 redirection status.
* Full support for handling https protocol to prevent errors in certain server configurations.
* Works without Apache .htaccess
* Delete and Editing Functions
* Support all special characters in non-english URLs.

== Installation ==

1. Upload 301 WP URL SEO Redirection to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add redirects on the Settings > SEO 301 page.

== Frequently Asked Questions ==

= Why would I want to use this instead of .htaccess? =

No coding require. Redirection is easy to put while making redirects from cPanel/htaccess.etc is much difficult for beinggers and newbies.

= How Seo 301 Meta is effective for SEO? (SEO Hacker) =

301 redirect, or also known as a permanent redirect, should be put in place to permanently redirect a page.
The word permanent is there to imply that ALL qualities of the redirected page will be passed on to the detour page.
* That includes:

** PageRank
** MozRank
** Page Authority
** Traffic Value

= What's difference between 301 and 302? =

* 301, Moved Permanently - recommended for SEO
* 302, Found or Moved Temporarily

= 301 Redirection Importance (MOZ SEO) =

A 301 redirect is a permanent redirect which passes between 90-99% of link juice (ranking power) to the redirected page.
301 refers to the HTTP status code for this type of redirect. In most instances, the 301 redirect is the best method for implementing redirects on a website.
