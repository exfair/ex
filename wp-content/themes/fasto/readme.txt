Fasto it's a lightweight minimal theme, built for speed and readability.

== Description ==
-Built with speed in mind - achieve perfect page speed on desktop and mobile. Tested with Google Page Speed Insights, GT Metrix and pingdom tools.
-Theme options that allow you to make customizations - change logo, colors, blog layout, social links, custom css code
-Four blog layouts - Grid with 4 per row, 3 per row, 2 per row with sidebar, classic with sidebar
-Basic optimization options - lazy load, and inline critical css
-Social links profiles for: Facebook, Twitter, Youtube, Linkedin, Pinterest, Dribbble, Instagram, Behance
-Custom widgets
-WooCommerce support
-Author section under post single
-Article social share
-Floating social share in article

By using default WordPress Customizer
